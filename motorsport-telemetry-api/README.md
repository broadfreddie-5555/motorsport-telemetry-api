# Motorsport Telemetry Projects

Backend-focused portfolio demonstrating API development, database interaction, and performance analysis using Python and Flask.

---

## Projects

### 1. Telemetry API
A REST API for storing and retrieving motorsport telemetry data.

Features:
- Store lap data (driver, sectors, tyre, speed)
- Retrieve telemetry records
- Fastest lap endpoint per track

---

### 2. Telemetry Comparison Engine
An analytics API that compares driver performance using lap time data.

Features:
- Compare drivers by average lap time
- Query telemetry sessions
- Performance insights via API endpoints

---
